### ‚ú® Usage

Using it is simple. Just type `8ball` followed by your question in quotes.

```bash
> 8ball "Should I deploy this on a Friday?"
> My sources say no.

> 8ball "Will I become a Python packaging expert?"
> Without a doubt.
```
